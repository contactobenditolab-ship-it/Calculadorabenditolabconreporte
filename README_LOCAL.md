# Abrir la calculadora sin restricciones (recomendado)

Cuando abres el archivo como `file://`, el navegador puede bloquear popups y algunas funciones.  
La solución es abrirlo desde `http://localhost` con un servidor local.

## Opción A (1 click): `start-local.bat`
1. Doble click en `start-local.bat`
2. Abre en el navegador: `http://localhost:8000/`

## Opción B (PowerShell): `start-local.ps1`
1. Click derecho → **Ejecutar con PowerShell**
2. Abre: `http://localhost:8000/`

## Qué URL abrir
- `http://localhost:8000/` (redirige a `index.html.html`)
- o directamente `http://localhost:8000/index.html.html`

## Parar el servidor
Vuelve a la ventana donde se está ejecutando y pulsa `Ctrl + C`.
